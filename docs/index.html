Installing crush

If you're on Windows, we recommend you the binary crush.exe. Run it as per our examples below. If you're on UNIX, you'll need to compile TestU01 first. For most systems, the following commands should suffice: $ unzip TestU01.zip [...] $ cd TestU01-1.2.3/ $ ./configure [...] $ make [...] $ sudo make install Now the TestU01 libraries are installed. To compile crush, tell your shell: wget http:.../ tar zxf crush-0.1.tar.gz cd crush-0.1 touch crush.c make make install Now crush is installed in your /usr/local/bin. To test your system's generator, tell your shell: $ cat /dev/urandom | crush -b small -n /dev/urandom [...]
