Introduction

Crush is a command-line program that lets you run your random number generator against the batteries of statistical tests from the TestU01 library with only using your shell.

Installing crush

If you're on Windows, we recommend you the binary crush.exe. If you're on UNIX, you'll need to compile TestU01 first. For most systems, the following commands should suffice: 
$ wget http://simul.iro.umontreal.ca/testu01/TestU01.zip
$ unzip TestU01.zip
$ cd TestU01-1.2.3/
$ ./configure
$ make 
$ sudo make install
Now the TestU01 libraries are installed. To compile crush, tell your shell: 
$ wget https://dbastos0.github.io/crush/files/crush-0.3.tar.gz
$ tar zxf crush-0.3.tar.gz
$ cd crush-0.3
$ make 
$ sudo make install
Now crush is installed in your /usr/local/bin. To test your system's generator, tell your shell: 
$ cat /dev/urandom | crush -b small -n /dev/urandom
